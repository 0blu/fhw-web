FHW Web Dokumentation
=====================

Hier findest du sämtliche für die Bearbeitung der Übung notwendigen Funktionen beschrieben,
die das FHW-Web Framework bereitstellt.

Für die Bearbeitung der Übung 01 genügt das Kapitel "Erste Schritte".

Nachfolgende Kapitel werden sukzessive hinzugefügt und aktualisiert. Aus Neugierde können neue Funktionalitäten
gerne ausprobiert werden; sollten aber bei der Bearbeitung der jeweiligen Übung nicht eingesetzt werden.
Für Übung 01 ist *ausschließlich* das Kapitel "Erste Schritte" maßgeblich.
